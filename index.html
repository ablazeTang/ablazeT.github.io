How to build a personal blog by hexo, node and git

1. install node.js, hexo, git

Download and install node.js:
  https://nodejs.org/en/download/

Check node.js version:
  execute the command ‘node -v’

Install hexo:
  npm install hexo -g
Check hexo:
  hexo -v

Download and install git:
  https://git-scm.com/
  Need to config the ssh-key

2. install dependency

  npm install –save hexo-deployer-git

3. new a blog dir and git repo

new a blog by hexo
mkdir blog
cd blog
hexo init
hexo g (generate a static web page)
hexo s (preview in localhost)

new a git repo
Create a new repo which named username.github.io
the username must be your github user name

4. Deploy the blog dir on github

  1. Change the config file _config.yml
    _config.yml under the dir blog/ needs to change, detail as follows:

    Set the deploy type to git
    Set the deploy repo
    Set the deploy branch

code note:

deploy:
type: git
repository: git@github.com:username/username.github.io.git
branch: main
  1. Deploy to github

    hexo d

  1. access the blog

    https://username.github.io

Some command for hexo

hexo new “postName” #new a article
hexo new page “pageName” #new a page
hexo generate #generate a static webpage under public
hexo server #start server on port 4000
hexo deploy #deploy blog to github
hexo help # for help
hexo version #for version
